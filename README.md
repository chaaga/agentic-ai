# Agentic AI

A personal collection of local-first AI/agentic projects and learnings —
built for hands-on practice with RAG, LLM tooling, and agent patterns.

## Projects

- **[rfp-rag](rfp-rag/)** — RAG-based RFP assistant. Drafts answers to new RFP
  questions by matching them against a repository of previously answered
  questions, reranking, and refining with a local LLM (Ollama). See
  [rfp-rag/README.md](rfp-rag/README.md) for setup and usage.

## Concepts

[docs/concepts.md](docs/concepts.md) is a running knowledge base of AI
engineering concepts (RAG fundamentals, RAG vs MCP/tools, agent patterns, and
more), accumulated while building the projects in this repo.
